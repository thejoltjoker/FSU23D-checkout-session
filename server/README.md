# Server

## Endpoints

### User

- GET /api/user
  - Get user information


- GET /api/user/orders
  - Get orders for logged in user

- POST /api/user/orders
  - Place an order

- PUT /api/user/orders
  - Update an order
  

### Auth

- POST /api/login

- POST /api/logout

- POST /api/register

- GET /api/authorize
